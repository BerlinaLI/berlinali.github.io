Key Word Search Bot

Client // Pei Enen
Role // browser interaction and interface
Tool // jQuery,Javascript,Chrome Plug-in

Key word search bot is a chrome plug-in that searches keywords automatically to manipulate the website's recommendation system. Rather than running on terminal, it provides an graphical user interface for general public to influence the website's algorithm easily. Only available on Chrome
