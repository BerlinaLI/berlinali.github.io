Sound Flappy Bird

Client //indivisual work
Role // front-end development
Tool // p5.js
Play the game by controlling your voice volume.
It is a web game to break the silence in the physical space. You can see it as an ice-breaker as well to get other people's attention (unconsciously).
The program is running on p5.js. Due to privacy constraint to use audio input, it is not avaiable on ios device.
