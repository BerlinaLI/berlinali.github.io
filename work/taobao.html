E-commerce Hacker Plug-in

Client // Pei Enen
Role // Front-end Development
Tool // jQuery, Chrome Plug-in
The recommendation algorithm of most online shopping websites is based on user's browsing history. By manipulating the history, we can change the way the websites target us.
By mimicking human's searching behavior to shop online, the robot/plug-in can input search keywords, select products, scroll for product details, even click on recommended products, and continue the loop endlessly.
Unlike most data scrapping project running on terminal, the robot runs in the front-end with jQuery to make the experience like a public performance in front of users. It is of user's interest to provide a list of search keywords whether is relevant to their real preference. This project is made possible in collaboration with Luciano Hanyon to provide technical support for artist Pie Enen.
